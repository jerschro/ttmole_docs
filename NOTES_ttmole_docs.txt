command to run website
ensure conda environment is running
    " mkdocs serve "
    or
    " python -m mkdocs serve "

required dependencies currently
    mkdocs
    mkdocstrings
    mkdocs-material
    mkdocstrings-python
    mkdocs-glightbox
" conda config --add channels conda-forge"
" conda install mkdocs mkdocstrings mkdocs-material mkdocstrings-python numpy"
" pip install mkdocs-glightbox"

how to upload a folder to github (all in powershell)
git init -b main
git add .
git commit -m "Commit Message"
git remote add origin https://github.com/jerschro/repository.git
#git remote add origin https://github.com/jerschro/ttmole_docs.git
git branch -M main
git push -u origin main

how to update github website (all in powershell)
do all previous commands
python3 -m mkdocs gh-deploy
